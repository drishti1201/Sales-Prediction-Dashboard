# Sales Prediction Dashboard using Machine Learning

## Project Overview

This project predicts future sales using Machine Learning and provides an interactive Streamlit dashboard for business analysis. It combines data preprocessing, exploratory analysis, model training, and real-time prediction.

## Features

- Interactive Streamlit Dashboard
- Sales Prediction
- Feature Importance Analysis
- Model Performance Comparison
- Business Summary Dashboard
- Machine Learning Model Deployment

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Joblib

## Machine Learning Models

- Random Forest Regressor
- Gradient Boosting Regressor

## Dataset

Superstore Sales Dataset

## Results

- High prediction accuracy
- Interactive dashboard
- Feature importance visualization
- Business insights from sales data

## Future Improvements

- XGBoost implementation
- Deep Learning model
- Cloud deployment
- Real-time database integration
